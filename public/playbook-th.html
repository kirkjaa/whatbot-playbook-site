MemorySkillsMCP/A2A SettingsModel Configuration

Stage: [Idea / MVP / Revenue / Growth / Fundraising]

v1.7 รวมทั้งโปรไฟล์หลักของ Agent Zero และโปรไฟล์เฉพาะของ Whatbot ให้เลือก lead agent ก่อน แล้วค่อย delegate งานเฉพาะทางเมื่อจำเป็น

Venture Builderไอเดีย การ validate, competitive intel, consulting, fundraising, scaling และ exit strategy
Financial Analystงานการเงินระดับ CFA ด้วย Fincept Terminal: valuation, portfolio, derivatives, risk, quant และ backtesting

Developerออกแบบระบบ เขียนโค้ด debug และใช้ code execution
Hackerความปลอดภัย penetration testing การตรวจ risk และ hardening

รักษาบริบทกลางร่วมกัน และบอกฉันเมื่อควรเปลี่ยน profile เพื่อให้ผลลัพธ์ดีขึ้น

ใช้ Multi-Agent Venture Consulting

เลือกได้ว่าจะให้ระบบ route อัตโนมัติด้วย smart mode, เรียก specialist เฉพาะคน หรือรันทั้ง 5 คนพร้อมกันด้วย turbo mode

venture-consultantMarket AnalystBusiness AnalystCompetitor AnalystFinancial AnalystPitch Consultant
Smart Modeโหมดแนะนำ ระบบเลือก specialist ที่เกี่ยวข้อง 1-3 คนอัตโนมัติ
Turbo Modeรันทั้ง 5 analysts พร้อมกันเพื่อมุมมองครบถ้วน
Single-Agent Modeระบุ `market_analyst`, `business_analyst`, `competitor_analyst`, `financial_analyst` หรือ `pitch_analyst`
Prompt สำหรับ Turbo Consulting 
Run a Venture Consultant review in turbo mode and answer in Thai.

Context:
[สรุปบริษัท สินค้า traction business model ตลาด และ funding goal]

Ask all five specialists to analyze:
1. Market size, growth, and timing
2. Business model and pricing
3. Competitive positioning and moat
4. Financial projections, unit economics, runway
5. Pitch narrative, investor objections, and funding readiness

End with a prioritized 30 / 60 / 90 day action plan.
Prompt สำหรับทดสอบมุมมองนักลงทุน 
ทำหน้าที่เป็นนักลงทุน Series A ที่สงสัยและเข้มงวด

Challenge การวิเคราะห์ล่าสุด:
- อะไรทำให้คุณปฏิเสธดีลนี้ทันที
- สมมติฐานใดต้องมี evidence ภายในเดือนนี้
- metric ใดยังขาดหรืออ่อน
- milestone ใดทำให้บริษัท fundable มากขึ้นชัดเจนที่สุด

ตอบอย่างตรงไปตรงมา เจาะจง และ quantify risk เท่าที่ทำได้

Prompt สำหรับ Investor Deck 
Use QUEST Studio `generate_doc`.

Document type: pitch_deck
Quality: hideft
Audience: Seed / Series A investors
Topic:
[สรุปบริษัท product traction market business model raise amount และ use of funds]

Style direction:
[สีแบรนด์ mood reference และข้อจำกัดด้านภาพ]

Create a 12-slide investor deck with problem, why now, solution, product, market, traction, business model, GTM, competition, team, financials, and ask. Use Thai language where appropriate. Save the editable file and preserve the PPTX download link.
เลือก quality: ใช้ `draft` เพื่อร่างเร็ว, `standard` สำหรับงานทั่วไป, `hideft` สำหรับนักลงทุน/ลูกค้า และ `hideft_max` สำหรับงาน final ระดับ board หรือ fundraising

Prompt สำหรับ Final Package 
รวบรวมทุกอย่างจากแชทนี้เป็น final venture operating package ภาษาไทย

Include:
1. Executive summary
2. Venture brief
3. Market and competitive analysis
4. Financial model summary
5. GTM strategy
6. Investor deck checklist
